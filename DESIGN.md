# Takwara Design System

> Instruções de identidade visual para agentes AI.
> Baseado no padrão Google DESIGN.md.

## Cores
- **Verde Takwara:** `#4a7c59` (principal) / `#2d5a3a` (escuro) / `#e8f0e4` (fundo claro)
- **Cinza texto:** tons #1f2937 (título), #4b5563 (corpo), #6b7280 (secundário), #9ca3af (discreto)
- **Fundo:** #faf6f0 (creme) / #0d1117 (dark mode)
- **Destaques:** #c084fc (roxo), #facc15 (ouro)

## Tipografia
- **Títulos:** Georgia, serif — peso 700
- **Corpo:** system-ui, sans-serif — peso 400
- **Código:** 'SF Mono', 'Fira Code', monospace

## Tom de voz
- Direto, sem enrolação
- Técnico mas acessível (engenheiro que conversa com artesão)
- Uso de perguntas retóricas para engajar
- Tabelas e bullet points para escaneabilidade

## Layout
- Largura máxima: 800px (texto), 960px (full)
- Sidebar esquerda com navegação hierárquica
- Badges de idioma no topo: PT 🇧🇷 / EN 🇺🇸 / ES 🇪🇸
- Rodapé com créditos e links para repositório

## Logo
- Local: `/images/takwara-logo.png`
- Uso: header do site, favicon
- Não distorcer, manter proporção 2:1
